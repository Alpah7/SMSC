# smsc.io
SMSC Open Source Solution with Monitoring, Billing, SMPP, SS7 and REST API support.

# Start admin module

## Installation
* `cd modules/admin`
* `npm install`

## Serve
* `npm start` 

> go to [http://0.0.0.0:3000](http://0.0.0.0:3000) or [http://localhost:3000](http://localhost:3000) in your browser
